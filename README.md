# OTW-Bandit-Scripts
Bash scripts for automating CTF challenges from OverTheWire (Bandit).

## Scripts
* 'bandit24sh.sh' - A multithreaded-like fast brute-force script for Level 24 of Bandit OverTheWire that solves a 4-digit PIN challenge through a single netcat connection.

### Usage 
1. Make the script executable:
# Enter in bash terminal:
# chmod +x bandit24sh.sh

2. Run the script:
# Enter in bash terminal:
# ./bandit24sh.sh

3. Enter the requested information(Password, pin size and the target port)
